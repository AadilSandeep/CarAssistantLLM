# Contributing

Thank you for your interest in contributing to the Car Diagnostic Assistant LLM project.

## Project Goal

The objective of this project is to explore the use of Large Language Models and automotive diagnostic knowledge for assisting users in understanding vehicle symptoms and OBD-II fault codes.

---

## Development Setup

Clone the repository:

```bash
git clone <repository-url>
cd Car_Assistant_LLM
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

---

## Contribution Guidelines

### Reporting Issues

When reporting issues, please include:

* Operating system
* Python version
* Error messages
* Steps to reproduce

### Pull Requests

Before submitting a pull request:

1. Ensure code runs successfully.
2. Maintain existing project structure.
3. Update documentation when necessary.
4. Verify model loading and inference still work.
5. Keep changes focused and well documented.

---

## Code Style

* Follow PEP 8 guidelines.
* Use descriptive variable names.
* Add comments where appropriate.
* Prefer modular and maintainable code.

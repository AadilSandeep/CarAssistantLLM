## [1.0.0] - 2026-06-15

### Added

* Microsoft Phi-2 based automotive diagnostic assistant
* LoRA fine-tuned adapter integration
* OBD-II diagnostic support
* Gradio web interface
* Local Apple Silicon deployment support
* Model verification script (`test_load.py`)
* OBD code interpretation utilities
* Structured diagnostic report generation

### Changed

* Migrated project from Google Colab to a local VS Code project structure
* Refactored notebook code into modular Python components

### Fixed

* Restored behavioral parity with original notebook implementation
* Fixed OBD prompt contamination issue
* Corrected OBD code interpretation workflow
* Optimized Apple Silicon inference configuration

### Documentation

* Added professional README
* Added MIT License
* Added repository metadata and project documentation
